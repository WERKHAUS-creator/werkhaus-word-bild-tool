# WERKHAUS Word Add-in

Verbindliche Manifeste:
- DEV: `manifests/manifest.dev.xml`
- PROD: `manifests/manifest.prod.xml`

Legacy-Manifestdateien im Projektroot sind veraltet und nicht mehr fuer Start/Validierung zu verwenden.

## Standardbefehle
- `npm install`
- `npm run dev-server`
- `npm run start:dev`
- `npm run validate:dev`
- `npm run validate:prod`
- `npm run build`
- `npm run lint`
